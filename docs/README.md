## Project Overview

The Shree Hindutakht platform is a social community application built with:
- **Backend**: Laravel PHP framework
- **Frontend**: Blade templates with Tailwind CSS
- **Mobile**: Android WebView application
- **Database**: SQLite/MySQL

## Key Features

### Member Features
- Social feed with text and image posts
- Event management and RSVP system
- Donation processing
- Profile management
- Digital ID card access
- Notification system

### Administrative Features
- Dashboard with statistics
- Member management
- Content moderation
- Event scheduling
- Donation tracking

## Recent Enhancements

### Image Processing
- Automatic image compression
- WebP conversion for smaller file sizes
- Target file size of 50KB or less

### Authentication
- Native Android app authentication
- Seamless login experience
- Instant routing without visible redirection

### Privacy
- Comprehensive privacy policy
- Account deletion process
- Data retention policies
- Legal compliance documentation
